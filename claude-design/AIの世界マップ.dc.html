第5部 エリア④ MCP港 — TOOLS

MCP港：4つの定番サービス

AIがつながる先の道具箱。開発・仕事で使う定番サービス群です。

GitHub

コードのバージョン管理・共有プラットフォーム。AIコーディングツールの共通の置き場所。

Notion

メモ・文書・タスク管理・DBを1つにまとめるワークスペース。AI連携も進む。

Slack

チーム向けチャットツール。チャンネル整理とAIボット連携での自動化が広がる。

Google Drive

Googleのクラウドストレージ。文書の保存・共同編集の基盤で、AI要約・検索の対象にもなる。

第6部 エリア⑤ クラウド王国 — CLOUD

クラウド王国：3大クラウド

データを保存し、AIを動かす計算資源を提供する基盤です。

AWS

Amazon

サーバー・DB・AI関連サービスを従量課金で利用でき、多くのAIサービスの裏側を支える。

Azure

Microsoft

OpenAIモデルの主要な提供経路の一つで、企業向けAI導入の中心的存在。

Google Cloud

Google

Gemini等の自社AIモデルとネイティブに統合され、データ分析・機械学習基盤としても利用。
